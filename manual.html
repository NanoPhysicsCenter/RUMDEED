RUMDEED’s documentation (Jun 07, 2024)

RUMDEED is a molecular dynamics program for simulations of vaccum diodes. It was developed at by the NanoPhysics group at Reykjavík University. RUMDEED stands for Reykjavík University Molecular Dynamics code for Electron Emission and Dynamics. It is an open-source software distributed under the terms of the GNU Public License Version 3 (GPLv3). The code can be obtained from the NanoPhysics group Github page | https://github.com/NanoPhysicsCenter/RUMDEED
