6. Code description¶

The code

6.1. Verlet¶

For particle advancement the code uses the Velocity Verlet algorithm. The code for it can be found in mod_verlet.F90 and the main subroutine is:

subroutine  velocity_verlet(step)¶

Velocity Verlet

Parameters

step [integer,in] :: Scaled in units of time_scale

Called from

update_position()

Call to

set_voltage(), update_elechole_position(), write_position_test(), calculate_acceleration_particles(), write_acceleration_test(), update_velocity(), write_ramo_current()

The updated positions are calculated with:

\[x_{n+1} = x_n + v_n\Delta t + \frac{1}{2}a_n \Delta t^2\]

Update the position of particles in the verlet integration

Then the force on each particle is calculate using Coulomb’s law, after which the velocity is update using:

\[v_{n+1} = v_n + \frac{a_n+a_{n+1}}{2} \Delta t^2\]

6.3. Field emission¶

Field emission

\[J = \frac{a}{\phi t^2(l)}F^2 exp(-\nu(l)b\phi^{3/2}/F)\]

See [5] for nu and t

\[\nu(l) = 1 - l + \frac{1}{6}l \ln(l)\]
\[t(l) = 1 + l\left( \frac{1}{9} - \frac{1}{18}\ln(l) \right)\]
\[l = \frac{F}{F_\phi} = \frac{e^3}{4\pi\epsilon_0} \frac{F}{\phi^2}\]

If (phi) is in eV and (F) in V/m then

\[l = \frac{e}{4\pi\epsilon_0} \frac{F}{\phi^2}\]
