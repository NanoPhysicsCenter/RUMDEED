4. Output and data files

Output files after running the code can be found in the out folder. Files either have an ending in .dt (Data Text) or .bin (Binary). Text file ending in .dt have to data orginzed as columns. Files ending in .bin are binary files. Both files types are decsribed below.

init.dt
The file init.dt contains the parameters used in the program. It is a Fortran namelist file were importand parameters are shown.
